# Mini 商城小程序

一个基于微信小程序原生语法构建的精简商城示例，包含首页营销、商品分类、商品详情、购物车与会员中心等基础功能模块，适合作为自定义电商小程序的起点模板。

## 功能亮点

- **数据模拟**：内置精选生鲜、零食与生活方式商品数据，支持分类与搜索过滤。
- **购物流程**：商品详情页支持轮播图、规格信息、加购与立即购买；购物车支持数量调节、合计金额与提交订单演示。
- **会员体验**：提供权益展示、常用功能入口以及服务指南，方便扩展更多会员场景。

## 项目结构

```
project.config.json     # 微信开发者工具项目配置
miniprogram/
├── app.js              # 全局数据与小程序实例
├── app.json            # 页面路由与 TabBar 配置
├── app.wxss            # 全局样式
├── data/products.js    # 商品与分类模拟数据
├── utils/cart.js       # 购物车本地存储工具
└── pages/
    ├── index/          # 首页
    ├── catalog/        # 商品分类与搜索
    ├── product/        # 商品详情
    ├── cart/           # 购物车
    └── profile/        # 我的页面
```

## 使用方式

1. 打开微信开发者工具，选择“导入项目”。
2. 将项目目录指向本仓库根目录（`/workspace/mini`）。
3. 在弹出的配置中填写或选择你自己的小程序 AppID（如仅预览可留空体验）。
4. 进入开发者工具后，即可在模拟器中预览并调试该商城示例。
